Code de Conduite Code of Conduct

Soyez impeccables les uns avec les autres. Si quelque chose n'est pas impeccable, parlez-en aux organisateurs. Be excellent to each other. If anything is less than excellent, talk to the organisers.

On s'attend à ce que vous vous teniez bien. C'est la bonne façon de passer un bon moment.

Cela dit, les organisateurs sont là. À défaut, il y a le personnel du [% PROCESS cn2 %].
Ne vous laissez pas embêter (pour rester poli), ne laissez pas quelqu'un se faire embêter.

We want to have a great time, so we expect you'll behave nicely. No need to insist on that.

Nevertheless there will be the organizers, the staff of [% PROCESS cn2 %].
Don't let anyone piss you off, don't let anyone being pissed off.
